# 🎧 Spotify Data Analytics Dashboard

## 📌 Overview
This project presents an interactive **Spotify Data Analytics Dashboard** built using Power BI.  
The dashboard provides insights into artist popularity, track trends, and album distribution using visually engaging charts and KPIs.

It demonstrates data visualization, storytelling, and analytical skills using real-world music streaming data.

---

## 📊 Key Features

### 📈 Artist Analysis
- Displays total number of artists
- Highlights most popular artists based on track metrics
- Top artists include Taylor Swift, The Weeknd, Ariana Grande, and more

### 🎵 Track Popularity Insights
- Identifies top-performing tracks such as *Runaway*, *Espresso*, and *Numb*
- Uses horizontal bar charts for easy comparison

### 💿 Album Distribution
- Visual breakdown of album types:
  - Albums
  - Singles
  - Compilations

### 🔢 Key Metrics (KPIs)
- **Total Artists:** 2549  
- **Total Albums:** 4832  

---

## 📌 Dashboard Components

- Bar charts for artist comparison  
- Horizontal bar charts for track popularity  
- Donut chart for album type distribution  
- KPI cards for quick insights  
- Dark-themed UI for enhanced visualization  

---

## 🛠️ Tools & Technologies Used

- **Power BI** – Data visualization and dashboard creation  
- **Power Query** – Data cleaning and transformation  
- **Data Modeling** – Relationships and DAX measures  

---

## 📈 Insights Derived

- A small group of top artists dominates overall popularity  
- Track popularity follows a skewed distribution (few hits, many low-performing tracks)  
- Albums make up the majority of content compared to singles and compilations
